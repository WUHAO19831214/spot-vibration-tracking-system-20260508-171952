# 已知局限

## 测量对象

系统测量红色反射光斑的纵向峰—峰范围 `Δy`，不是振动物体本体的机械振幅 `A`。反射镜转角、传播距离和入射角会改变光斑位移放大率，当前未反演这些因素。

## 颜色追踪

- 固定阈值不适应所有摄像头和照明；
- 背景红色区域会影响加权重心；
- 过曝光斑、散斑和拖影会导致偏移；
- 没有连通域、ROI、形态学或模型分类；
- 没有滤波和离群点剔除；
- 高分辨率逐像素 JavaScript 处理可能降帧。

## 时间与采样

- `requestAnimationFrame` 不是固定采样频率；
- 未使用摄像头硬件时间戳；
- 后台标签页会降频；
- `setInterval` 会受主线程负载影响；
- 频率切换瞬态未自动舍弃；
- 帧率不足会漏掉极值并低估 `Δy`。

## 音频与驱动

- 频率是 Web Audio 设定值，不是独立实测值；
- 普通扬声器无法可靠产生接近 0 Hz 的输出；
- 音量滑块不代表恒定机械驱动力；
- 驱动器和装置频率响应会混入结果；
- 没有闭环幅度控制。

## 空间标定

- 单一 1 cm 线段只给出局部线性比例；
- 标尺与光斑不共面会产生误差；
- 透视和镜头畸变未校正；
- 拖线端点误差进入比例因子；
- 相机或装置移动后必须重新标定。

## 数据逻辑

- 同频重复时只保留最大 `Δy`，会产生向上偏倚；
- 未标定时可能记录 0 cm；
- 不保存每个窗口的原始 `y(t)`；
- 没有不确定度、置信区间或质量标志；
- 刷新页面后数据丢失；
- 平滑曲线可能造成高于数据分辨率的视觉印象。

## 未实现

FFT、相位差、数字滤波、阻尼参数拟合、真实机械振幅反演、数据导出和自动实验报告均未实现。

## 建议

固定曝光/对焦/白平衡，使用无红色干扰背景；先验证帧率和驱动频率；每点停留多个周期并预留瞬态；保存原始视频；论文中同时报告失败案例。未完成验证前，将结果描述为教学演示或探索性数据。
